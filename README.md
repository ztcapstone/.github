# How to make Constitution/Bylaw/Policy changes

These documents are written in LaTeX. 

To edit them, download the code from this repository and paste it in your LaTeX editor of choice.

I recommend: https://www.overleaf.com/project

Make your changes...

Then make a GitHub branch

Copy your new code and paste it to the appropriate document on GitHub

Finally, make a pull request and merge the changes with the existing documents.
